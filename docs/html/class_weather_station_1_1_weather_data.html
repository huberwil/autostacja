Detailed Description

Agreguje pomiary pogodowe i oblicza statystyki dla stacji pomiarowej.

Główny magazyn danych systemu. Przechowuje historię pomiarów i udostępnia metody analityczne do przetwarzania szeregów czasowych.

◆ GenerateRandomReadings()

void WeatherStation.WeatherData.GenerateRandomReadings ( int count)

Generuje określoną liczbę losowych pomiarów symulujących pracę czujnika.

Parameters
countLiczba pomiarów do wygenerowania.

Pomiary są rozłożone losowo w czasie ostatnich 30 dni z rozdzielczością godzinową. Wartości temperatury mieszczą się w pełnym zakresie czujnika. Metoda służy do wypełnienia bazy danych przykładowymi danymi w celach demonstracyjnych i testowych.

◆ GetAverageTemperatureLastDays()

float WeatherStation.WeatherData.GetAverageTemperatureLastDays ( int days)

Oblicza średnią temperaturę z określonego okresu wstecz od chwili obecnej.

Parameters
daysLiczba dni wstecz stanowiących okres analizy.
Returns
Średnia arytmetyczna temperatury [°C] z okresu.

Algorytm obliczenia:

  1. Wyznacz graniczną datę na podstawie aktualnego czasu minus days .
  2. Przefiltruj kolekcję pomiarów wybierając tylko te z datą późniejszą niż granica.
  3. Zsumuj wartości wszystkich pasujących pomiarów.
  4. Podziel sumę przez liczbę pasujących pomiarów.
