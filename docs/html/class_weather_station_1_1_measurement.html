Detailed Description

Niezmienny rekord pojedynczego pomiaru wykonanego przez czujnik.

Klasa jest immutable — wszystkie pola są inicjalizowane w konstruktorze i nie mogą być modyfikowane. Zapewnia to bezpieczeństwo wielowątkowe oraz deterministyczne zachowanie w testach jednostkowych.
